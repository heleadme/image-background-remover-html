# 图片去背景工具 - MVP 需求文档

## 📋 项目概述

**项目名称**：图片去背景工具  
**版本**：v1.0 (MVP)  
**目标**：提供一个简单、快速、高质量的在线图片去背景服务

---

## 🎯 产品目标

### 核心价值
- ✅ **简单**：无需注册，上传即处理
- ✅ **快速**：秒级处理速度
- ✅ **高质量**：使用 Remove.bg API，专业级效果
- ✅ **免费**：提供免费额度，降低使用门槛

### 用户痛点
1. PS 太复杂，普通用户不会用
2. 其他在线工具要么慢，要么效果差
3. 需要下载安装软件，麻烦
4. 免费工具加水印或限制太多

## 🔧 技术方案

### 架构
- **前端**：纯 HTML/CSS/JavaScript（单文件）
- **API**：Remove.bg API (https://api.remove.bg/v1.0/removebg)
- **部署**：Cloudflare Pages
- **存储**：无后端存储，API Key 存 localStorage

### 技术选型理由
| 选型 | 理由 |
|------|------|
| 纯前端 | 简单、快速、无需维护后端 |
| Remove.bg API | 效果好、稳定、有免费额度 |
| Cloudflare Pages | 免费、全球 CDN、自动 HTTPS |
| localStorage | 简单、无需数据库 |

### API 集成
- **端点**：`POST https://api.remove.bg/v1.0/removebg`
- **认证**：Header `X-Api-Key`
- **请求**：FormData (`image_file`, `size=auto`)
- **响应**：Binary PNG
